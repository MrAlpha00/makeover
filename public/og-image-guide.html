How to Create the OG Image

This is a preview of how your Open Graph image should look. To create the actual image:

Option 1: Use Canva (Recommended)

  1. Go to canva.com and create a free account
  2. Search for "Social Media Post" template
  3. Select size 1200 x 630 pixels
  4. Create a design matching the preview above
  5. Download as JPG and save as public/og-image.jpg

Option 2: Use Figma

  1. Create a new frame at 1200 x 630px
  2. Design your OG image
  3. Export as JPG at 2x resolution
  4. Save as public/og-image.jpg

Option 3: Use Vercel OG Image

You can also use Vercel's built-in OG image generation by creating an API route.

Important: Your current public/og-image.jpg file may not exist. You need to create it manually using one of the methods above. The image should be exactly 1200x630 pixels for best results.
