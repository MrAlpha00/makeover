## 🚀 Quick Start

### Prerequisites
- **Node.js** 18+ installed
- **npm** or **yarn** package manager
- **Supabase** account (free tier)
- **Resend** account for email (free tier: 3,000 emails/month)

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/party-hub.git
cd party-hub

# Install dependencies
npm install

# Copy environment variables
cp .env.local.example .env.local

# Configure environment variables
# Edit .env.local with your Supabase and Resend credentials

# Start development server
npm run dev

# Open http://localhost:3000
```

### Database Setup

```bash
# Run database migrations/seeding
npm run seed
```

## 🎯 Supabase Database Setup

### Required Tables

```sql
-- Categories table
CREATE TABLE categories (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  image TEXT,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Subcategories table
CREATE TABLE subcategories (
  id SERIAL PRIMARY KEY,
  category_id INTEGER REFERENCES categories(id),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  sort_order INTEGER DEFAULT 0,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Designs/Services table
CREATE TABLE designs (
  id SERIAL PRIMARY KEY,
  category_id INTEGER REFERENCES categories(id),
  subcategory_id INTEGER REFERENCES subcategories(id),
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  description TEXT,
  images TEXT[],
  base_price DECIMAL,
  discounted_price DECIMAL,
  inclusions TEXT[],
  addons JSONB,
  tags TEXT[],
  is_featured BOOLEAN DEFAULT FALSE,
  is_active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP DEFAULT NOW()
);

-- Bookings table
CREATE TABLE bookings (
  id SERIAL PRIMARY KEY,
  customer_name TEXT NOT NULL,
  customer_email TEXT,
  customer_phone TEXT NOT NULL,
  service_id INTEGER REFERENCES designs(id),
  occasion_type TEXT,
  event_date DATE,
  venue_address TEXT,
  guest_count INTEGER,
  message TEXT,
  status TEXT DEFAULT 'new',
  created_at TIMESTAMP DEFAULT NOW()
);

-- Settings table
CREATE TABLE settings (
  id SERIAL PRIMARY KEY,
  key TEXT UNIQUE NOT NULL,
  value TEXT,
  updated_at TIMESTAMP DEFAULT NOW()
);
```

## 🎨 Customization

### Update Contact Information
Replace `XXXXXXXXXXX` (WhatsApp number with country code, no +) in:
- `components/WhatsAppButton.js`
- `app/services/[slug]/page.js`
- `app/booking/page.js`
- `app/contact/page.js`
- `app/layout.js` (schema markup)
- `components/Navbar.js`
- `components/Footer.js`

### Update Site Name
Edit `NEXT_PUBLIC_SITE_NAME` in environment and update:
- `app/layout.js` (title, description, og:title)
- `components/Navbar.js` (logo text)
- `components/Footer.js` (brand name)

### Change Theme Colors
Edit `tailwind.config.js` to modify:
- Primary color (currently coral `#f95738`)
- Dark theme backgrounds
- Animation timings
